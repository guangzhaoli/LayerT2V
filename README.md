## Authors

Guangzhao Li, Kangrui Cen, Baixuan Zhao, Yi Xin, Siqi Luo, Guangtao Zhai, Lei Zhang, Xiaohong Liu

## Overview

LayerT2V is a unified multi-layer framework for video generation. The project focuses on layer-aware video synthesis, aiming to provide a flexible formulation for generating, composing, and controlling videos through multiple visual layers.

The upcoming release will include the implementation of the proposed framework, inference and training utilities, pretrained model checkpoints, dataset resources, and evaluation scripts.

## Highlights

- **Unified multi-layer formulation** for video generation.
- **Layer-aware generation pipeline** for structured video synthesis and composition.
- **Research-friendly implementation** with reproducible configuration files, scripts, and pretrained checkpoints.
- **Dataset and evaluation support** for reproducing the main experimental results.
